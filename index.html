Before you play it: Beat Saber wants the audio as .ogg. This tool bundles your original file as-is — if it isn't already .ogg, convert it (Audacity, or any free online converter) and swap the file inside the folder before dropping it into CustomLevels.
Maps are written in Beat Saber's v3 format, including arcs (long holds, used when a beat's gap to the next hit is notably longer than a single tap) and chains (a head note plus a string of linked segments, used for tight tremolo-style bursts on Expert/Expert+). No walls or bombs yet. Open the map in ChroMapper to fine-tune, add walls, or touch up any awkward patterns — the detected BPM and grid carry over cleanly.
Tempo detection uses a comb-filter search over the song's onset energy, so it works best on tracks with a steady, audible beat. Very ambient or rubato songs may need the manual BPM override.
